Passe as informações do pássaro que você quer cadastrar, é importante passar todas as informações, cuide para passar os dados corretamente, conforme o que você pesquisou, no momento não recebemos imagens no nosso servidor, então passe a url dela, considerando que você a salvou em algum sistema de nuvem, como google drive ou mega.

Após isso, clique no botão "cadastrar" para inserir ele no nosso banco de dados.

Caso não queira mais cadastrar o passaro, é só clicar em "cancelar" e você volta para a página principal.
